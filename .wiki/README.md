# GoCMS Wiki

Практические гайды для операторов и разработчиков. Материалы пополняются по мере типовых вопросов.

## Оглавление

| Раздел | Описание |
|--------|----------|
| [Onboarding 101](./onboarding-101.md) | Быстрый обзор платформы и первого запуска |
| [Архитектура и слои](./architecture/README.md) | Четыре части продукта и слои codex / stack / ui8kit |
| [Первый вход и учётные записи](./onboarding/README.md) | Цикл онбординга: безопасный запуск, логин, пароли, recovery |
| [Модель безопасности](./security/README.md) | Пароли, recovery, сессии, аудит на уровне принципов |
| [Аутентификация](./authentication/README.md) | Политики входа, локальный режим, playground, токены |

## Как пользоваться

1. Начните с [Onboarding 101](./onboarding-101.md), если ещё не поднимали окружение.
2. Перед первым боевым контуром пройдите [рекомендуемый цикл первого входа](./onboarding/recommended-first-login-cycle.md).
3. Остальные темы (миграции, бэкапы, health, аудит) будут вынесены в отдельные файлы по мере появления вопросов — их можно запросить в чате с привязкой к фрагменту Wiki.
